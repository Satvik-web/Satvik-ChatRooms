# Satvik-ChatRooms
Satvik ChatRooms is a peer-to-peer meeting room. People can join the meeting using the other person's Connection ID. 
Features: Audio and Video, White board, Screen Sharing, File Transfer, Chat. All the features are purely peer-to-peer no middle-men involved.

Check it out: [https://satvik-web.github.io/Satvik-ChatRooms/](https://satvik-web.github.io/Satvik-ChatRooms/)
